# Welcome to MD2Slide

> Live-preview and export Markdown as stunning web-based slides.

Press the **Preview Slides** button in the status bar or the toolbar icon to start.

---

## Slide Splitting

Use `---` on its own line to split content into separate slides.

- Each section becomes a slide
- Navigate with arrow keys
- Supports full Markdown syntax

---

## Code Example

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

---

## Export

Run `MD2Slide: Export to Standalone HTML` from the Command Palette to save your slides as a self-contained `.html` file.
