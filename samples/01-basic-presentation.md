# Welcome to MD2Slide 🎉

Turn your **Markdown** into beautiful presentation slides.

---

# How It Works

1. Write Markdown as usual
2. Separate slides with `---` on its own line
3. Open the preview panel (`Ctrl+Shift+M`)
4. Navigate with arrow keys or the outline sidebar

---

# Markdown You Know & Love

- **Bold** and *italic* text
- `inline code` snippets
- [Hyperlinks](https://revealjs.com)
- Images and tables
- And everything else Markdown supports

# Code Snippets

```typescript
// TypeScript example
function greet(name: string): string {
  return `Hello, ${name}! 👋`;
}

console.log(greet("World"));
```

```python
# Python example
def fibonacci(n: int) -> list[int]:
    a, b = 0, 1
    result = []
    for _ in range(n):
        result.append(a)
        a, b = b, a + b
    return result
```
